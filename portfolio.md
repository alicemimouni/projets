## Présentation

J'ai conçu le protfolio d'Esteban, celui-ci désirant que ces créations soient consultables sur le web par les recruteurs.

## Technologies utilisées

J'ai utilisé Symfony pour ce projet en fullstack. Esteban souhaitait modifier lui même son contenu, ses expériences, compétences, réalisations ...
Symfony m'a permis de créer le site web accompagné d'un tableau de bord personnalisé, de cette façon Esteban disposait d'une interface claire et simplifiée, pour gérer son site sans risque.
