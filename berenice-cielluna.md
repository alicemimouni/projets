## Présentation

Conception d'un site web pour Bérénice Cielluna, autrice à Saint-Etienne.
Bérénice souhaitait mettre en avant ses écrits avec un site web.
Elle souhaitait également un espace pour ajouter des livres et modifier son contenu.

## Technologies utilisées

Maquette réalisée avec Figma.

La cliente souhaitait un espace d'administration facile à prendre en main, j'ai donc utilisé Symfony pour lui concevoir un site web et son espace de gestion personnalisé.
Cela permet d'avoir un site web sobre, léger, facile à maintenir et une interface ultra personnalisée pour une prise en main facile.
